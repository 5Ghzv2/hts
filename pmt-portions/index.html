HTML

What is HTML?
  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating Web pages
  • HTML describes the structure of a Web page
  • HTML consists of a series of elements
  • HTML elements tell the browser how to display the content
  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

HTML Editors
  • Web pages can be created and modified by using professional HTML editors.
  • However, it can be created by simple editors like Notepad too.
    • While using Notepad to edit HTML documents, remember to save the file as .html and open it with a web browser.

Tags
  • Tags are the building blocks of a web page.
  • They contain elements that define how the information on a webpage is formatted or displayed.
  • We can also include non-text items, such as images, links, and lists on the web page using the text.
  • Tags do not appear in the browser window but they affect the display of the text and non-text items in it.
  • The HTML tags or commands are inserted before and after the text. Each tag will begin with a less than sign "<" and end with a greater than sign ">".
    • ex: <p> - On Tag, </p> - Off tag.

Attributes
  • An attribute is the property that provides additional information about an HTML element.
  • It enhances the functionality of a tag. An attribute is always specified inside the opening tag.
  • An attribute consists of two parts — a name and value.
  • The name is the property that we want to set and value is what we want for the respective property.
  • For example, the width attribute of the <hr> tag specifies the width of the horizontal line in pixels or percentage.
    • <hr width=50%>

Elements
  1. Container Elements
  2. Empty Elements

Container Elements
  • The element that include both on and off tags are called Container Elements.
    • ex: <b> this makes the text bold </b>

Empty Elements
  • Empty elements contain only on tags. They do not have off tags. These elements do not enclose any data.
    • ex: <br>

Overview of HTML Tags

Tag Defenition
<br> This tag is an example of an empty element, where you do not need opening and closing tags, as there is nothing to go in between them. </br> element, anything following it starts from the next line.
<center> This tag is used to put any content in the center of the page or any table cell.
<hr> The <hr> tag creates a line from the current position in the document to the right margin and breaks the line accordingly.
<i> This tag makes the content Italicize.
<b> This tag makes the cotent bold.
<u> The tag underlines the content given.
<strike> This tag strikethrough the text.
<sup> The font size used is the same size as the characters surrounding it but is displayed half a character's height above the other characters.
<sub> The font size used is the same as the characters surrounding it, but is displayed half a character's height beneath the other characters.
<table>

Creates A Table

Attributes:

  1. bgcolor: You can set background color for whole table or just for one cell.
  2. background: You can set background image for whole table or just for one cell.
  3. border: We can set the border for the table using the border attribute.
  4. bordercolor: We can also set border color.
  5. width: We can set a table width.
  6. height: We can set a table height.
<tr> Create table rows
<td> Create data cells
<th> Inserting table heading. Datas are centered and bold by default
<ul>

An unordered list. This will list the items using plain bullets.

Attributes:

  1. Type: We can use type attribute for <ul> tag to specify the type of bullet you like.
    • a) square b) disc c) circle
    • <ul type="square">
    • <ul type="disc">
    • <ul type="circle">
<ol>

An ordered list. This will use different schemes of numbers to list your items.

Attributes:

  1. Type:
    • <ol type="1"> - Default-Case Numerals.
    • <ol type="I"> - Upper-Case Numerals.
    • <ol type="i"> - Lower-Case Numerals.
    • <ol type="A"> - Upper-Case Letters.
    • <ol type="a"> - Lower-Case Letters.
<font>

to add style, size, and color to the text on your website.

Attributes:

  1. size: We can set content font size using size attribute. The range of accepted values is from 1(smallest) to 7(largest).
  2. face: We can set font face.
  3. color: We can set any font color.
Linking Documents [<font>]

A link is specified using HTML tag <a>. This tag is called anchor tag and anything between the opening <a> tag and the closing </a> tag becomes part of the link and a user can click that part to reach to the linked document.

Syntax:

  • <a href = "Document URL" ... attributes-list>Link Text</a>

ex:

  • <a href = "/html/index.htm">Opens in New</a>
<img>

You can insert any image in your web page by using tag. Following is the simple syntax to use this tag.

  • <img src = "Image URL" ... attributes-list/>

The <img> tag is an empty tag, which means that, it can contain only list of attributes and it has no closing tag.
<html> This tag encloses the complete HTML document and mainly comprises of document header which is represented by <head>...</head> and document body which is represented by <body>...</body> tags.
<head> This tag represents the document's header which can keep other HTML tags like <title>, <link> etc.
<title> The <title> tag is used inside the <head> tag to mention the document title
<body>

This tag represents the document's body which keeps other HTML tags like <h1>, <div>, <p> etc.

Attributes:

  1. bgcolor: to apply background color for the webpage.
